# Code of Conduct

AgentReady follows the Contributor Covenant Code of Conduct.

## Our Pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be kind and direct.
- Assume good intent while still addressing real issues.
- Keep criticism focused on the work.
- Respect maintainers' time and project scope.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Dismissive or demeaning language.
- Posting private information without permission.
- Spam, mass-generated issues, or low-effort promotional PRs.

## Enforcement

Maintainers may remove comments, close issues, block accounts, or take other reasonable steps to protect the project.

Report concerns by opening a private security advisory or emailing the maintainer listed on the GitHub profile.
